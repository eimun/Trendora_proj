# UI Refinements

## Dark Mode
- Tailwind CSS class-based dark mode (`darkMode: 'class'`)
- Theme toggle in sidebar with localStorage persistence
- All components support light/dark themes

## Animations (Framer Motion)
- Card hover effects with scale transitions
- Staggered loading animations for trend cards
- Modal enter/exit animations with AnimatePresence
- Smooth sidebar collapse/expand

## Data Visualization (Recharts)
- AreaChart for historical trend volume
- Animated progress bars for gap scores
- Confidence bars with gradient fills

## Landing Page
- Semrush-inspired marketing page with 6 sections
- Animated competitor comparison bars
- Role-based tabs with animated underline

## Sidebar Navigation
- Collapsible icon sidebar (72px → 240px)
- Active route purple accent bar
- Tooltip on hover when collapsed
- Grouped sections (Core / Advanced)
