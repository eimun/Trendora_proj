# Trendora — Deployment Plan

## Architecture Overview

```
┌─────────────────┐      ┌─────────────────────┐      ┌──────────────┐
│   Vercel CDN    │      │   Render Web Service │      │  Render DB   │
│   (Frontend)    │─────▶│   (Flask + Gunicorn) │─────▶│ (PostgreSQL) │
│   React SPA     │ API  │   trendora-api       │      │ trendora-db  │
└─────────────────┘      └─────────────────────┘      └──────────────┘
                                    │
                         ┌──────────┴──────────┐
                         │                     │
                  ┌──────▼──────┐      ┌───────▼──────┐
                  │ Render      │      │ Render       │
                  │ Worker:     │      │ Worker:      │
                  │ Archiver    │      │ Predictor    │
                  │ (Daily)     │      │ (Daily)      │
                  └─────────────┘      └──────────────┘
```

## Services

### 1. Frontend — Vercel

| Setting | Value |
|---------|-------|
| **Platform** | Vercel |
| **Framework** | Create React App |
| **Build Command** | `npm run build` |
| **Output Directory** | `build/` |
| **Root Directory** | `frontend/` |
| **Environment Variable** | `REACT_APP_API_URL` = Render backend URL |

**Config file:** `frontend/vercel.json`

---

### 2. Backend API — Render Web Service

| Setting | Value |
|---------|-------|
| **Platform** | Render |
| **Service Type** | Web Service |
| **Runtime** | Python 3.11 |
| **Build Command** | `pip install -r requirements.txt` |
| **Start Command** | `gunicorn app:app` |
| **Root Directory** | `backend/` |

**Environment Variables:**
| Variable | Source |
|----------|--------|
| `DATABASE_URL` | Auto-linked from Render PostgreSQL |
| `GEMINI_API_KEY` | Manual (Google AI Studio) |
| `YOUTUBE_API_KEY` | Manual (Google Cloud Console) |
| `SECRET_KEY` | Auto-generated by Render |
| `PYTHON_VERSION` | `3.11.0` |

## Deployment Steps

### Initial Setup

1. **Push code** to GitHub repository
2. **Render:**
   - Import `render.yaml` via Dashboard → Blueprints
   - Set `GEMINI_API_KEY` and `YOUTUBE_API_KEY` in environment
   - Database auto-provisions via blueprint
3. **Vercel:**
   - Import frontend folder from GitHub
   - Set `REACT_APP_API_URL` to Render backend URL
   - Deploy

### Redeployment

- **Auto-deploy** on push to `main` branch (both Render and Vercel)
- **Manual:** Trigger via Render/Vercel dashboards
